## What is NISR?

**NISR** is a **multilingual, beginner-friendly programming language** designed to **break language barriers in coding** by enabling people‚Äîespecially across Africa and underserved communities‚Äîto **learn and create using their native languages**.

More than just a tool, **NISR** is a **movement to make programming universally accessible, inclusive, and culturally relevant**. By combining **localized learning** with **lightweight tools**, it **empowers children, students, and beginners‚Äîeven in rural areas‚Äîto solve real-world problems, foster innovation, and become part of a diverse global developer community**.


This guide helps new users quickly learn how to write, compile, and run their first NISR programs.


---

## üìÇ 1. Create Your First NISR File

### **1.1 Create the File**
All NISR source files must use the `.ns` extension.

- **File Extension:** `.ns`
- **Example Filename:** `main.ns`

### **1.2 Open the File**
Open your newly created `.ns` file in a code editor such as **VS Code**.
